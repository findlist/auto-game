## 具体修改内容

### 1. `src/index.css` 全局样式增强

#### 1.1 CSS 变量系统扩展
新增精细层次变量，统一阴影、圆角与玻璃质感：

```css
--shadow-sm: 0 2px 8px rgba(0, 0, 0, 0.08);
--shadow-lg: 0 8px 30px rgba(0, 0, 0, 0.18);
--shadow-glow: 0 0 24px rgba(102, 126, 234, 0.35);
--radius-sm: 8px;
--radius-lg: 16px;
--tube-highlight: rgba(255, 255, 255, 0.55);
--tube-shadow-inner: rgba(0, 0, 0, 0.18);
--color-layer-shine: rgba(255, 255, 255, 0.45);
```

#### 1.2 字体与背景优化
- 字体栈调整为 `'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei'`，提升中文渲染质感
- 新增 `background-attachment: fixed`，背景渐变随滚动固定
- 新增 `font-feature-settings: 'tnum' on, 'cv11' on`，数字等宽对齐
- `-webkit-tap-highlight-color: transparent` 去除移动端点击高亮

#### 1.3 首页视觉提升
- logo 增加 `filter: drop-shadow(0 6px 12px rgba(0,0,0,0.25))` 光晕
- title 增加 `letter-spacing: 2px` 字间距 + 多层 text-shadow 立体感

#### 1.4 试管玻璃质感
`.tube` 增强玻璃材质表现：

```css
box-shadow:
  inset 2px 0 4px rgba(255,255,255,0.25),
  inset -2px 0 4px var(--tube-shadow-inner);
```

新增 `.tube::before` 伪元素，绘制左侧高光线条，模拟玻璃反光。

#### 1.5 颜色层立体球体效果
`.color-layer` 通过多层 inset box-shadow + 径向渐变高光实现立体球体感：

```css
.color-layer {
  box-shadow:
    inset 0 2px 3px var(--color-layer-shine),
    inset 0 -2px 3px rgba(0, 0, 0, 0.15),
    inset 2px 0 3px rgba(255, 255, 255, 0.15),
    inset -2px 0 3px rgba(0, 0, 0, 0.08);
}

.color-layer::before {
  /* 径向渐变高光，模拟球体反光 */
  background: radial-gradient(ellipse at center, var(--color-layer-shine) 0%, transparent 70%);
}
```

#### 1.6 动画过渡优化
- 倾倒动画 `pour-bounce` 改用 `cubic-bezier(0.34, 1.4, 0.64, 1)` 增加回弹效果
- `layer-fade-in` 动画增加 scale 变化，让入场更有层次
- 新增 `current-pulse` 动画，当前关卡按钮呼吸式光晕

#### 1.7 按钮交互反馈
`.btn` 系列增强交互反馈：

```css
.btn:hover {
  transform: translateY(-2px);
  filter: brightness(1.08);
}
.btn:active {
  transform: translateY(0) scale(0.98);
}
```

#### 1.8 关卡按钮质感
- `.level-btn` 增加内阴影质感
- `.level-btn.current` 新增 `current-pulse` 呼吸动画
- 难度标记改为 `border-left`，颜色更突出

#### 1.9 游戏面板层次
`.game-board` 增强：
- 使用 `--shadow-lg` 加深阴影
- 新增 `::before` 顶部高光线条

#### 1.10 移动端响应式优化
- `.tubes-grid` padding 改为 `16px 4px`，避免边缘贴边
- `@media (max-width: 375px)` 优化试管间距和触摸区域
- 底部导航 `@media (max-width: 480px)` 横向滚动优化

#### 1.11 信息页面与卡片
- `.info-page h3` 新增 `border-left` 增强层次
- `.stat-card` 新增 `::before` 顶部高光
- `.achievement-card` 增强 hover 效果和已解锁状态光晕

### 2. `src/components/TubeView.tsx` 试管视图

颜色层添加纵向渐变 `backgroundImage`，增强立体球体感：

```tsx
backgroundImage: 'linear-gradient(180deg, rgba(255,255,255,0.28) 0%, rgba(255,255,255,0.05) 40%, rgba(0,0,0,0.12) 100%)'
```

顶部高光、中部过渡、底部暗影，配合 CSS 的 inset box-shadow 与 `::before` 径向高光，共同呈现立体球体效果。

### 3. `src/components/GameBoard.tsx` 游戏主面板

回放试管颜色层与 TubeView 保持一致的纵向渐变效果，确保回放视图与实际游戏视觉统一。

### 4. `src/pages/LevelEditorPage.tsx` 关卡编辑器

两处颜色层渲染（编辑器试管、自定关卡游玩器）都添加了相同的纵向渐变，保持全局视觉一致性。

## 总结

本次样式优化在不改变任何功能逻辑的前提下，通过以下手段全面提升游戏视觉品质：

1. **建立完整的设计令牌系统**：三级阴影、三级圆角、玻璃质感变量
2. **强化核心视觉元素**：试管玻璃质感、颜色层立体球体效果
3. **补全缺失样式**：约 400 行编辑器和自定关卡样式
4. **优化交互反馈**：按钮、卡片、关卡按钮的 hover/active 状态
5. **完善移动端体验**：响应式断点优化、触摸区域优化、横向滚动适配
6. **统一视觉语言**：所有颜色层渐变效果全局一致
