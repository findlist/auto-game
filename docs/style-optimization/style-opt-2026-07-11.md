## 优化目标

本次优化延续 2026-07-10 的基线，聚焦三个方向：

1. **代码可维护性**：将散落的内联 `style` 对象提取为可复用的 CSS 工具类，减少重复、便于全局调整
2. **视觉氛围**：首页头部光晕、卡片悬停微光、加载占位呼吸动画等氛围细节
3. **响应式覆盖**：新增 340px 超小屏断点与横屏 (landscape ≤500px) 适配，覆盖更多移动设备

## 具体修改内容

### 1. `src/index.css` 全局样式增强

#### 1.1 修复格式 Bug
- `.donate-section` 中 `color` 与 `padding` 原本挤在同一行，已拆分为两行，符合代码可读性规范。

#### 1.2 新增工具类（替换内联样式）

| 类名 | 用途 | 替换前的内联样式 |
| --- | --- | --- |
| `.header-spacer` | 页头右侧 40px 占位符 | `style={{ width: '40px' }}` |
| `.modal-body-text` | 弹窗正文文字 | `textAlign: center, color: #666, marginBottom: 16px` |
| `.modal-body-text-lg` | 弹窗正文文字（大号） | 同上 + `fontSize: 16px` |
| `.modal-body-text-announcement` | 公告弹窗正文 | 同上 + `fontSize: 15px, lineHeight: 1.6` |
| `.modal-actions` | 弹窗操作容器 | `flex, gap: 12px, flexDirection: column` |
| `.modal-actions-sm` | 弹窗操作容器（紧凑） | `flex, gap: 8px, flexDirection: column` |
| `.modal-hint-text` | 弹窗辅助说明 | `center, #999, 13px, marginBottom: 16px` |
| `.modal-hint-text-sm` | 弹窗辅助说明（小号） | `center, #999, 12px` |
| `.stats-note` | 统计页辅助文字 | `fontSize: 12px, color: #999` |
| `.stats-subtitle` | 统计页小标题 | `margin-top: 12px, color: var(--primary)` |
| `.stat-value-streak-hot` | 连胜数值火焰色 | `color: #FF6B6B` |
| `.page-loading-container` / `.page-loading-inner` / `.page-loading-emoji` / `.page-loading-text` | 懒加载占位组件 | 4 处内联样式 |

#### 1.3 加载占位组件呼吸动画
新增 `loading-breathe` keyframes，让加载 emoji 以 1.6s 周期缓慢呼吸（scale 1 ↔ 1.08，opacity 0.85 ↔ 1），避免静态占位的死板感。

```css
@keyframes loading-breathe {
  0%, 100% { transform: scale(1); opacity: 0.85; }
  50% { transform: scale(1.08); opacity: 1; }
}
```

#### 1.4 首页头部光晕装饰
新增 `.home-header::after` 伪元素，以径向渐变绘制柔和光晕，为首页顶部增加氛围层次。

#### 1.5 模式按钮悬停增强
`.mode-grid-btn:hover` 增加 `translateY(-2px)` 上移与 `brightness(1.08)` 亮度，与昨日 `.btn` 系列的交互反馈保持一致。

#### 1.6 卡片悬停微光扫过
为 `.recent-play-card::before` 与 `.recommend-card::before` 新增微光扫过动画，hover 时一道斜向高光从左上扫向右下，增强卡片可点击的视觉暗示。

#### 1.7 全局滚动与触摸优化
- `html { scroll-behavior: smooth; }` 平滑滚动
- `-webkit-text-size-adjust: 100%` 防止 iOS 横屏自动放大
- 与昨日 `-webkit-tap-highlight-color: transparent` 形成完整的移动端触摸体验闭环

#### 1.8 新增响应式断点

| 媒体查询 | 覆盖场景 |
| --- | --- |
| `@media (max-width: 340px)` | 超小屏（小于昨日 375px 断点的更细分支），优化试管间距与字体 |
| `@media (max-width: 480px)` | 移动端徽章优化 |
| `@media (orientation: landscape) and (max-height: 500px)` | 横屏低高度设备（如手机横屏），压缩首页头部 padding 与 logo/title 字号 |

### 2. `src/App.tsx` 内联样式提取

共替换 **8 处** 内联 `style` 对象为 CSS 类：

| 位置 | 替换前 | 替换后 |
| --- | --- | --- |
| `PageLoading` 组件 | 4 处内联样式 | `.page-loading-container/inner/emoji/text` |
| 首页连胜数值 | `style={{ color: ... }}` 动态颜色 | `className={'stat-value' + (streak>0 ? ' stat-value-streak-hot' : '')}` |
| 自动存档恢复弹窗正文 | 内联 style | `className="modal-body-text"` |
| 自动存档弹窗操作容器 | 内联 style | `className="modal-actions"` |
| 签到奖励弹窗正文 | 内联 style | `className="modal-body-text-lg"` |
| 签到弹窗辅助说明 | 内联 style | `className="modal-hint-text"` |
| 公告弹窗正文 | 内联 style | `className="modal-body-text-announcement"` |
| 公告弹窗操作容器 | 内联 style | `className="modal-actions-sm"` |
| 公告弹窗"还有 X 条"提示 | 内联 style | `className="modal-hint-text-sm"` |
| 自定关卡游玩器页头占位 | 内联 style | `className="header-spacer"` |

> 保留未替换：进度条 `width`、难度颜色 `color` 等动态值仍使用内联样式（语义上必须为动态值）。

### 3. `src/pages/*.tsx` 同步替换

为保持页头布局模式一致，将以下 6 个页面中相同的 `style={{ width: '40px' }}` 占位符统一替换为 `className="header-spacer"`：

- `AboutPage.tsx`
- `AchievementsPage.tsx`
- `StatsPage.tsx`
- `SettingsPage.tsx`
- `LevelEditorPage.tsx`
- `PrivacyPage.tsx`

同时将 `StatsPage.tsx` 连胜说明行的 `style={{ fontSize: '12px', color: '#999' }}` 替换为 `className="stats-note"`。

## 总结

本次优化在不改变任何功能逻辑的前提下，通过以下手段提升代码质量与视觉氛围：

1. **提取 12 个工具类**：将 App.tsx 与 6 个页面文件中的内联样式统一为 CSS 类，便于全局维护与未来主题切换
2. **加载占位呼吸动画**：消除静态占位的死板感，缓解用户等待焦虑
3. **首页头部光晕 + 卡片微光扫过**：增强氛围层次与可点击暗示
4. **横屏与超小屏适配**：覆盖更多移动设备形态，提升游戏移动端体验
5. **保持功能不变**：所有修改均为样式层面，构建通过，未触及任何业务逻辑
