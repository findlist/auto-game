## 关键约束说明
`src/game/themeManager.ts` 在运行时通过 `document.documentElement.style.setProperty` 动态覆盖以下 8 个核心 CSS 变量：
`--bg-gradient`、`--tube-bg`、`--tube-border`、`--primary`、`--secondary`、`--text`、`--text-light`、`--card-bg`

因此本次优化**避免在 `:root` 中修改这些变量的默认值**，而是：
1. 新增不会被覆盖的静态辅助变量
2. 增强静态 CSS 效果（玻璃质感、弯月面、光晕、阴影、动画）
3. 将硬编码的品牌色（`#667eea` / `#764ba2`）替换为 `var(--primary)` / `var(--secondary)`，让 UI 随主题自适应

## 具体修改内容

### 1. 新增静态辅助变量（index.css `:root`）
新增 6 个不会被 themeManager 覆盖的变量：
- `--tube-highlight-right`：试管右侧次高光颜色
- `--tube-bottom-reflection`：试管底部反光颜色
- `--meniscus-shine`：液体表面弯月面高光
- `--meniscus-shadow`：液体表面暗影
- `--glass-rim`：玻璃边缘反光
- `--bg-atmosphere`：背景氛围光晕渐变
- `--theme-transition`：主题切换过渡时长（0.4s ease）

### 2. 背景氛围层（index.css）
- 新增 `.app::before` 固定定位的氛围光晕层，在背景渐变之上叠加柔和的径向光晕，增加空间纵深感
- `.app > *` 提升 z-index 确保内容在氛围层之上

### 3. 主题切换平滑过渡（index.css）
- `body` 添加 `transition: background, color`，主题切换时背景与文字平滑过渡，避免突兀闪烁
- `.game-board` 添加 `transition: background, box-shadow`，卡片背景平滑变化
- `.tube` 的 transition 增加 `background` 和 `border-color`，试管玻璃随主题平滑切换

### 4. 试管玻璃质感增强（index.css `.tube`）
**优化前**：仅左侧单条高光 + 内阴影
**优化后**：
- `box-shadow` 新增底部反光（`inset 0 -3px 6px var(--tube-bottom-reflection)`），增强试管底部的立体感
- 新增 `.tube::after` 右侧次高光线条（1.5px 宽，从顶部渐变到透明），与左侧主高光呼应，形成完整的玻璃管反光效果

### 5. 液体弯月面效果（index.css）
**优化前**：颜色层顶部无液体表面表现
**优化后**：
- 新增 `.tube-inner > .color-layer:last-child::after`，仅在顶层颜色层显示弯月面高光
- 使用 `border-radius: 50% 50% 0 0 / 100% 100% 0 0` 模拟液体表面弧形
- 顶部 3px 高的白色渐变模拟液体表面反光，增强液体真实感

### 6. 颜色层立体感增强（index.css `.color-layer`）
**优化前**：box-shadow 内高光 2px / 内阴影 2px
**优化后**：box-shadow 内高光增至 4px、内阴影增至 4px，左右渐变也增强，让液体层球体感更饱满，液体沉降感更强

### 7. 选中试管发光主题自适应（index.css）
**优化前**：`.tube-container.selected .tube` 和 `@keyframes selected-glow` 硬编码 `rgba(102, 126, 234, ...)`（经典紫主题色），切换到暗夜黑/霓虹光等主题时发光色不匹配
**优化后**：使用 `color-mix(in srgb, var(--primary) 50%, transparent)` 让发光颜色随 `--primary` 主题变量自适应，所有 6 套主题的选中发光都与主题色一致

### 8. 粒子效果发光（ParticleEffect.tsx）
**优化前**：圆形粒子使用纯 `fillStyle` 填充，无发光效果
**优化后**：
- 圆形粒子添加 `ctx.shadowBlur = p.size * 2` 和 `ctx.shadowColor = p.color`，产生光晕扩散效果
- 拖尾部分重置 `shadowBlur = 0` 保持清晰
- 每帧结束后重置 `ctx.shadowBlur = 0` 防止影响其他粒子

### 9. 主按钮主题自适应（index.css `.btn-primary`）
**优化前**：`background: linear-gradient(135deg, #667eea, #764ba2)` 硬编码，阴影也是硬编码紫色
**优化后**：`background: linear-gradient(135deg, var(--primary), var(--secondary))`，`box-shadow` 使用 `color-mix` 自适应，按钮配色随主题变化

### 10. 全局品牌色统一替换为主题变量（index.css）
将以下硬编码的 `#667eea` / `#764ba2` 渐变统一替换为 `var(--primary)` / `var(--secondary)`：
- `.page-btn` 背景渐变
- `.level-badge` 背景渐变
- `.progress-bar-fill` 背景渐变（`--success` → `--primary`）
- `.progress-overall-bar` 背景渐变
- `.trend-bar` 背景渐变（180deg）
- `.loading-spinner` border-top-color
- 所有 `linear-gradient(135deg, #667eea, #764ba2)` 实例（共 11 处）

**注意**：`.diff-normal` 相关的语义化难度颜色保留硬编码不变，因为它们代表难度等级而非主题。
