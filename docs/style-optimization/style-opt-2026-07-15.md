## 具体修改内容

本次优化在 `src/index.css` 末尾追加"2026-07-15 样式优化增强（第三批）"区块，共 19 项增强，全部为追加样式，不修改已有规则。

### 1. 背景氛围层（`.app::after`）
- 新增两个柔和漂浮的径向光晕（左上 + 右下）
- 通过 `ambient-float` 18 秒缓动循环，营造空间纵深感
- `pointer-events: none` + `z-index: 0` 确保不干扰交互

### 2. 首页标题渐变文字（`.home-header .title`）
- 标题文字应用 `linear-gradient` + `background-clip: text`，形成白→淡蓝→淡粉的渐变
- 使用 `@supports not (-webkit-background-clip: text)` 提供回退方案
- 保留 `drop-shadow` 兼容旧浏览器

### 3. 副标题淡入上浮（`.subtitle`, `.tagline`）
- 0.15 秒延迟的 `header-fade-up` 动画
- 与主标题形成节奏感，避免首屏一次性闪现

### 4. 试管液体液面波动（`.color-layer:last-child::after`）
- 3.5 秒周期的 `liquid-ripple` 动画
- 通过 `scaleX` 微弱变化（1 → 1.08）模拟液面呼吸
- 让静态试管也有"液体活着"的观感

### 5. 试管悬停光晕（`.tube-container:hover .tube`）
- 悬停时叠加 12px 的白色外发光
- 强化"可交互"的视觉提示

### 6. 模式按钮过渡优化（`.mode-grid-btn`）
- 使用 `cubic-bezier(0.34, 1.3, 0.64, 1)` 弹性曲线
- hover 时上浮 3px + 缩放 1.02 + 亮度 1.05
- 阴影从默认提升到 `0 8px 24px`

### 7. 首页统计数字脉动（`.home-stats .stat-value`）
- hover 时缩放至 1.08
- 连胜数字（`.stat-value-streak-hot`）持续 2.2 秒脉动循环

### 8. 快捷导航卡片图标弹跳（`.quick-nav-card:hover .quick-nav-icon`）
- 悬停时图标缩放 1.15 + 上浮 2px

### 9. 页面加载占位优化（`.page-loading-*`）

## 技术要点

1. **纯 CSS 实现**：所有增强通过 CSS 变量、伪元素、动画完成，无 JS 改动
2. **渐进增强**：使用 `@supports` 和 `@media` 查询提供回退方案
3. **性能优化**：
   - 动画使用 `transform` 和 `opacity`，避免触发重排
   - `pointer-events: none` 确保装饰元素不拦截事件
   - `will-change` 提示浏览器优化合成层
4. **主题一致性**：所有颜色通过 CSS 变量引用，主题切换时自动适配
5. **无障碍优先**：
   - 焦点环与品牌色统一
   - 尊重 `prefers-reduced-motion`
   - 暗色主题独立适配
